Ayuda

Bienvenido al manual de creación de presentaciones, aquí puedes consultar los comandos para crear una presentación desde un archivo ".txt", todo el material a consultar lo puedes encontrar a continuación para que puedas hacer tu primera presentación desde cero.

Sobre los archivos ".txt"

Un archivo ".txt" es un archivo de texto plano, dónde se puede escribir lo que tu quieras en formato ASCII, la idea de estos archivos es la escritura de texto legible por humanos, en este caso, para crear una presentación deberás crear un archivo de estos. A continuación te mostraremos cómo hacer tu primera presentación con un archivo ".txt".

Crear un archivo ".txt"

Puedes crear un archivo .txt en cualquier lugar de tu computadora que tengas a la mano, para ello, puedes hacer derecho en cualquier parte de tu pantalla y presionas el botón de "Nuevo" y consiguiente te aparecerá un submenú que muestra qué archivos puedes crear. Para nosotros nos interesa el que dice "Documento de Texto" y haciendo clic en esta opción te creará un archivo ".txt". Despúes de esto, simplemente puedes ponerle el nombre que te guste y pulsas la tecla de "Enter".

Crear una presentación

Para crear una presentación puedes escribir en el archivo ".txt" para poder crear la presentacion, en este caso puedes escribir el texto de la presentación como texto y esté será y hacer uso de los distintos elementos se usan distintos comandos para varias cosas.

Simbología de los elementos de una diapositiva

Agregar una diapostiva

Para crear una diapositiva nueva puedes escribir [ingresar simbolo(s)], esto te permitirá definir cuándo empieza una nueva diapostiva. Si no se especifica ningún símbolo todo se agregará a una simple diapositiva. (No es recomendable si hay mucho texto en tu archivo)

Hacer un título

Para definir un título en una diapositiva usa [ingresar simbolo(s)], este símbolo hará que todo lo de una línea sea un título, por ejemplo: [ingresar simbolo(s)] Esto de aquí es un título

Escribir texto normal

Para empezar a escribir texto normal, puedes simplemente escribirlo sin símbolo despúes de definir una diapositiva. Por ejemplo:
[Simbolo de Diapositiva] <- A partir de aquí todo texto sin símbolos será cuerpo del texto
[Simbolo de Título] Esto es un título
Todo esto es texto de la diapositiva.

Cambiar color de Texto

Puedes usar los símbolos [ingresar simbolo(s)] para encerrar una palabra o varias palabras para cambiar el color del texto, el color del texto se debería de colocar como alguno de los siguientes:
  1. "Red" = Color Rojo
  2. "Blue" = Color Azul
  3. "Green" = Color Verde
  4. "Yllw" = Color Amarillo
  5. "Black" = Color Negro
  6. "Orge" = Color Naranja
  7. "Brwn" = Color Cafe

Hacer un subtítulo

Para definir un subtítulo en una diapositiva usa [ingresar simbolo(s)], similar a cómo funciona un título ya que hace que todo lo de una línea sea un subtítulo, por ejemplo: [ingresar simbolo(s)] Esto de aquí es un subtítulo
